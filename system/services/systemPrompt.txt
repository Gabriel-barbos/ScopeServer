# CONHECIMENTO — DISPOSITIVO MHUB

Dispositivo com identificação de motorista via crachá e liberação de ignição via relé.

**Funcionamento:**
1. Chave gira → painel liga
2. Leitor ativa e emite alerta sonoro
3. Motorista passa crachá
4. Sistema valida e libera partida
5. Dados enviados: ID do crachá, ID do motorista, ID do veículo, status (ligado/desligado)

**Problemas comuns:**

Falta de comunicação
- Área de sombra → normal, dados enviados depois via buffer
- Longo período sem comunicação → verificar alimentação, possível falha de instalação; persistindo: manutenção

Leitor travado (não lê crachá / não desliga / veículo bloqueado)
1. Desativar leitor via Mhub Diag
2. Aguardar comando chegar → libera partida
3. Encaminhar para manutenção

Dispositivo travado (crachá reconhecido mas não libera ignição)
1. Desabilitar função do leitor
2. Enviar reset
3. Persistindo: troca / manutenção

Nossa plataforma de gerenciamento de frota com dados dos veículos em tempo real.

**Problemas mais comuns:**
