# CONHECIMENTO — DISPOSITIVO MHUB

Descrição:

Dispositivo com:
	•	Identificação de motorista via crachá
	•	Liberação de ignição via relé

Funcionamento:
	1.	Chave gira → painel liga
	2.	Leitor ativa e emite alerta sonoro
	3.	Motorista passa crachá
	4.	Sistema valida e libera partida
	5.	Dados enviados:
	•	ID do crachá
	•	ID do motorista
	•	ID do veículo
	•	Status (ligado/desligado)

⸻

# PROBLEMAS COMUNS — MHUB

1. Falta de comunicação
	•	Pode ocorrer por área de sombra (normal)
	•	Dados são enviados posteriormente (buffer)

⚠️ Anormal:
	•	Longo período sem comunicação

Ações:
	•	Verificar alimentação
	•	Possível falha de instalação
	•	Caso persistente: encaminhar para manutenção

⸻

2. Leitor travado

Sintoma:
	•	Não lê crachá
	•	Não desliga
	•	Veículo bloqueado

Solução:
	1.	Desativar leitor via Mhub Diag
	2.	Aguardar comando chegar

Resultado:
	•	Liberação da partida

Ação posterior:
	•	Encaminhar para manutenção

⸻

3. Dispositivo travado

Sintoma:
	•	Crachá reconhecido, mas não libera ignição

Solução:
	1.	Desabilitar função do leitor
	2.	Enviar reset

Se persistir:
	•	Encaminhar para troca/manutenção
